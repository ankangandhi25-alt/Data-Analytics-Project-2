# Data Analytics Project 2

## DecodeLabs Data Analytics Internship

### Project Objective
Perform Exploratory Data Analysis (EDA) on a retail sales dataset using Microsoft Excel.

### Analysis Performed
- Data Cleaning
- Summary Statistics
- Product Analysis
- Payment Method Analysis
- Referral Source Analysis
- Data Visualization
- Business Insights

### Key Insights
- Total Orders: 1200
- Average Quantity: 2.95
- Average Unit Price: ₹356.41
- Average Order Value: ₹1053.97
- Printer received the highest number of orders.
- Online was the most used payment method.
- Instagram generated the highest referrals.

### Tools Used
- Microsoft Excel
